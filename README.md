# MaryaGetu(); — Personal Portfolio Website

A modern, responsive personal portfolio website built for Marya Getu, a Computer Science and Engineering student. The site showcases her skills, projects, and blog posts with a sleek dark-themed UI, interactive elements, and smooth animations. It is designed to serve as a professional online presence for developers, designers, and creatives.

## ✨ Features

- **Fully Responsive** – Mobile‑first layout that adapts seamlessly to all screen sizes.
- **Interactive Hero Section** – Typing animation, waving hand, and glowing profile image.
- **Skills Tabs** – Switch between Tech Stack and Tools with animated transitions.
- **Portfolio Grid** – Display projects with hover effects and direct links to GitHub.
- **Blog Section** – Expandable cards with "Read More" toggle for longer posts.
- **Smooth Navigation** – Sticky header with active link highlighting and smooth scrolling.
- **Social Links** – Connect via LinkedIn, Email, GitHub, and Telegram.
- **Glassmorphism Design** – Blurred backgrounds, subtle borders, and gold accent colors.
- **Dynamic Background Orb** – Subtle animated gradient for visual depth.

## 🛠️ Tech Stack

- **HTML5** – Semantic markup
- **CSS3** – Custom properties (variables), animations, and responsive design
- **JavaScript** – Tab switching, active link handling, read‑more toggles
- **Font Awesome 6** – Icons for skills, social links, and UI elements
- **Google Fonts** – Poppins font family
